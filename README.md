# Survey_Data_Analysis
Attached is snippet of code created to evaluate and visualize the data from a survey for smart thermostats.
To run this code you will need the following libraries:
- Pandas (install using pip install pandas)
- Matlibplot (install using pip install matplotlib)
- JSON (no need to install, just import)
- os (no need to install, just import)
- re (no need to install, just import)
